# CS300 Project Two - ABCU Course Planner

C++ program for SNHU CS300 that:
- Loads course data from a CSV file into a Binary Search Tree (BST)
- Prints all courses in alphanumeric order
- Prints detailed info for a selected course, including prerequisite titles
- Follows the Project Two rubric using one .cpp file

## How to Run
cd ~/Desktop/CS300-ProjectTwo
g++ ProjectTwo.cpp -o ProjectTwo
./ProjectTwo

## Files
- ProjectTwo.cpp  (main program)
- courses.csv     (input file)
